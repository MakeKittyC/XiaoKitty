 第三条 数据管理
 3.1 数据存储
- 所有用户生成内容（如文档、编辑记录）仅保存于设备本地，默认不采用加密
- 系统备份不受限制，用户可通过手机自带的云服务备份应用数据，一般来说此软件包程序不会产生任何重要的应用数据，只需备份软件包即可
- 数据可以存放 cache 目录，也就是缓存目录，也可以存放 file 目录，也就是文件目录，存放缓存目录的主要目的就是为了方便用户清理应用数据，直接清理所有数据会导致已授权的权限需要重新授权，当然，用户也可以使用 shell 执行器清理掉一些数据目录，在清理之前，应确保数据目录中没有重要数据或不存在重要数据，否则数据丢失将无法找回
